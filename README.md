# 📡 TVWS 가용채널 및 기기 운용 현황 대시보드

TVWS(TV White Space) 장비의 사용 이력 통계(CSV) 데이터를 분석하고, 기기 상태 및 위치를 시각화하는 Streamlit 기반의 대시보드입니다.

## 🌟 주요 기능

1. **장비 상태 자동 분류**
   - 🟢 **정상**: 24시간 이내 통신 이력이 있는 장비
   - 🟠 **주의**: 24시간이 경과한 장비 (26년 이후 데이터 기준)
   - 🔴 **미사용**: 2026년 이전 데이터로 현재 미사용 중인 장비 별도 분류
2. **Master-Slave 연결 거리 계산**
   - 하버사인(Haversine) 공식을 이용해 각 기기와 통신상대방(Master-Slave) 간의 직선거리(km) 자동 계산
3. **지도 시각화**
   - `Folium`을 사용하여 장비의 위경도 좌표를 기반으로 지도에 마커 표시
   - 상태(정상/주의/미사용)에 따라 마커 색상 구분
